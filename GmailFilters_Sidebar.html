🗂️ Gmail Filters
Bulk Filter Management

📥 Step 1: Pull Filters

Fetch all existing filters from your Gmail account into this spreadsheet.

📤 Step 2: Push Changes

Create, Update, or Delete filters based on the "Action" column in the sheet.

Instructions:

  • Set Action to CREATE for new filters.
  • Set Action to UPDATE to modify existing ones.
  • Set Action to DELETE to remove them.
  • Label names are case-sensitive.
